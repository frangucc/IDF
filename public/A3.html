Q. Solve the following problems by using substitution.

If y is an even integer and x is an odd integer, which of the following expressions could be an even integer?

  1. 3x + 
     y 
     2 

  2.  x + y 
     2 

  3. x + y

  4.  x 
     4 
     − 
     y 
     2 

  5. x2 + y2

Solution:Choose x = 1 and y = 2. Then 3x + 
 y 
 2 
 = 3 × 1 + 
 2 
 2 
 = 4 , which is even. The answer is (A).
Note: We don’t need to check the other answer-choices because the problem asked for the expression that could be even. Thus, the first answer-choice that turns out even is the answer.
