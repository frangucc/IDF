SYMBOL & REPRESENTATION

In this type of analogy, one word stands for or represents a concept, action, or thing. An example will illustrate.

Example:
CARET : INSERT ::

  1. colon : sever

  2. pie : exponentiate

  3. gun : lance

  4. period : stop

  5. scalpel : delete

Solution:A CARET (^) is an editing symbol that indicates where a word should be INSERTED. Similarly, a PERIOD is a grammatical symbol indicating a STOP. The answer is (D).

Example:
OLIVE BRANCH : PEACE ::

  1. key : extraction

  2. Trojan horse : deception

  3. icon : beautification

  4. directory : enumeration

  5. abbreviation : abridgment

Solution:An OLIVE BRANCH stands for PEACE. Similarly, a TROJAN HORSE stands for DECEPTION. The answer is (B).
