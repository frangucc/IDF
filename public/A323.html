Solution: Multiplying both sides of 3 < y < 7 by –1 yields –3 > –y > –7. Now, we usually write the smaller number on the left side of an inequality. So –3 > –y > –7 becomes –7 < –y < –3. Add this inequality to the like inequality –3 < x < –1:

–3 < x < –1

(+) –7 < –y < –3

–10 < x – y < –4

Dividing –10 < x – y < –4 by 2 yields
