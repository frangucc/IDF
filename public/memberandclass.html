MEMBER AND CLASS

Member and class analogies are usually easy problems. In this type of analogy, the first word may be an element of the class that the second word describes, or vice versa. This category can also be classified as "Type Of."

Example:
LIMERICK : POEM ::

  1. monologue : chorus

  2. sonnet : offering

  3. waltz : tango

  4. skull : skeleton

  5. aria : song

Solution:A LIMERICK is a type of POEM. Similarly, an ARIA is a type of SONG. The answer is (E). The eye-catcher is SONNET since it is a type of POEM. However, there is no relationship between SONNET and OFFERING.
