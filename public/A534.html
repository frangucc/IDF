Q.   The positive integers P, Q, R, S, and T increase in order of size such that the value of each successive integer is one more than the preceding integer and the value of T is 6. What is the value of R?

  1. 0

  2. 1

  3. 2

  4. 3

  5. 4

Solution:We know that T is 6; and therefore from the fact that "each successive integer is one more than the preceding integer" we see that S is 5. Continuing in this manner yields the following unique sequence:

P

Q

R

S

T

2

3

4

5

6

Hence, the value of R is 4. The answer is (E).
