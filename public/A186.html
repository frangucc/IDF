Q. Determine which of the two expressions below is larger, whether they are equal, or whether there is not enough information to decide.

x2 − 7x + 10 = 0

Column A
10

Column B
x2

  1. Column A is larger

  2. Column B is larger

  3. Column A and Column B are equal

  4. Not enough information to decide.

Solution: Factoring the equation x2 − 7x + 10 = 0 yields

(x – 5)(x – 2) = 0

x – 5 = 0 or x – 2 = 0

x = 5 or x = 2

If x = 5, then x2 = 52 = 25 and Column B is larger. If x = 2, then x2 = 22 = 4 and Column A is larger. This is a double case.
Hence, the answer is (D).
