Q.   If x = (x + 2)x, for all x, what is the value of x + 2  − x − 2 ?

  1. –2

  2. x + 4

  3. 0

  4. x2

  5. 8(x + 1)

Solution:x + 2  â€“ x − 2
= ([x + 2] + 2)[x + 2] − ([x − 2] + 2)[x − 2]

= (x + 4)[x + 2] − x[x − 2]

= x2 + 6x + 8 − (x2 − 2x)

= x2 + 6x + 8 − x2 + 2x

The answer is (E).
