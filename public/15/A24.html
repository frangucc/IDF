Q. Determine which of the two expressions below is larger, whether they are equal, or whether there is not enough information to decide.

For all numbers x, x denotes the value of x3 rounded to the nearest multiple of ten.

Column A
x + 1

Column B
x + 1

  1. Column A is larger

  2. Column B is larger

  3. Column A and Column B are equal

  4. Not enough information to decide.

Solution:Suppose x = 0. Then x + 1 =  0 + 1 = 1 = 0, and x + 1 = 0 + 1 = 0 + 1 = 1. In this case, Column B is larger.
Next, suppose x = 1.
Then x + 1 = 1 + 1 = 2 = 10, and
x + 1 = 1 + 1 = 0 + 1 = 1.
In this case, Column A is larger. The answer is (D).
