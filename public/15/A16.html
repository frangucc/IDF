Q. Determine which of the two expressions below is larger, whether they are equal, or whether there is not enough information to decide.

x > 0

Column A
x2 + 2

Column B
x3 − 2

  1. Column A is larger

  2. Column B is larger

  3. Column A and Column B are equal

  4. Not enough information to decide.

Solution:Since x > 0, we need only look at x = 1, 2, and 1/2. If x = 1, then x2 + 2 = 3 and x3 − 2 = −1. In this case, Column A is larger.
Next, if x = 2, then x2 + 2 = 6 and x3 − 2 = 6. In this case, the two columns are equal. This is a double case and therefore the answer is (D).
