Q. Determine which of the two expressions below is larger, whether they are equal, or whether there is not enough information to decide.

x ≠ 0, –1 and x > 1

Column A
 1 
 x 

Column B
 1 
 x + 1 

  1. Column A is larger

  2. Column B is larger

  3. Column A and Column B are equal

  4. Not enough information to decide.

Solution: Now, x < x + 1; and since x > 1, it is positive.
Hence, dividing both sides of x < x + 1 by x(x + 1) will not reverse the inequality:

 x 
 x(x + 1) 
 < 
 x + 1 
 x(x + 1) 

Canceling yields
 1 
 x + 1 
 < 
 1 
 x 

The answer is (A).

Method II:

Since x > 1, it is positive and so is x + 1.
Hence, we can multiply both columns by x(x + 1). This yields

Column A

Column B

x + 1

x

In this form, it is clear that Column A is larger. The answer is (A).
