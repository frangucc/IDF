Q. Determine which of the two expressions below is larger, whether they are equal, or whether there is not enough information to decide.

x is an integer greater than 1.

Column A
x + x3

Column B
x4

  1. Column A is larger

  2. Column B is larger

  3. Column A and Column B are equal

  4. Not enough information to decide.

Solution: Since x is greater than 1, we need only plug in the numbers x = 2, 3, and 4. If x = 2, then x + x3 = 2 + 23 = 10 and x4 = 24 = 16. In this case, Column B is larger.
Next, if x = 3, then x + x3 = 3 + 33 = 30 and x4 = 34 = 81. In this case, Column B is again larger.
Next, if x = 4, then x + x3 = 4 + 43 = 68 and x4 = 44 = 256. In this case, Column B is once again larger.
Hence, the answer is (B).
