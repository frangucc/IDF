Q. In the standard coordinate system, which of the following points is the greatest distance from the origin:

  1. (–4, –1)

  2. (–3, 3)

  3. (4, 0)

  4. (2, 3)

  5. (0, 4)

Solution: Using the distance formula to calculate the distance of each point from the origin yields

d = √(−4)2 + (−1)2 = √17

d = √(−3)2 + (3)2 = √18

d = √(4)2 + (0)2 = √16

d = √(2)2 + (3)2 = √13

d = √(0)2 + (4)2 = √16
The answer is B.
