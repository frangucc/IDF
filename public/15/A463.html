Q.   If x2 + y2 = xy, then (x + y)4 =

  1. xy

  2. x2y2

  3. 9x2y2

  4. (x2 + y2)2

  5. x4 + y4

Solution:Adding 2xy to both sides of the equation x2 + y2 = xy yields

x2 + y2 + 2xy = 3xy

[ from the formula (x + y)2 = x2 + y2 + 2xy]
(x + y)2 = 3xy

Squaring both sides of this equation yields

(x + y)4 = 3xy2 = 9x2y2
The answer is (C).
