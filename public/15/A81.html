Q.   If
 x + y 
 x − y 
 = 3 and x and y are integers, then which one of the following must be true?

  1. x is divisible by 4

  2. y is an odd number

  3. y is an even integer

  4. x is an even number

  5. x is an irreducible fraction

Solution:Solution:
 x + y 
 x − y 
 = 3 . Multiplying both sides of this equation by (x – y) yields

x + y = 3(x – y)

x + y = 3x – 3y

–2x = –4y

x = 2y

Since we have expressed x as 2 times an integer, it is even. The answer is (D).
