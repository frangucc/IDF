Q.   Let a, b, and c be three integers, and let a be a perfect square. If a/b = b/c, then which one of the following statements must be true?

  1. c must be an even number

  2. c must be an odd number

  3. c must be a perfect square

  4. c must not be a perfect square

  5. c must be a prime number

Solution:Cross multiplying the equation a/b = b/c yields

ac = b2

Dividing by a yields c = b2/a

We are given that a is a perfect square.
Hence, a = k2, for some number k. Replacing a in the bottom equation with k2, we get c = b2/k2 = (b/k)2. Since we have written c as the square of a number, it is a perfect square. The answer is (C).
