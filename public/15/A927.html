Q. Choose the lettered pair that best expresses a relationship similar to that expressed in the original pair.

ISLAND : ATOLL ::

  1. peninsula : archipelago

  2. fire : spring

  3. hand : glove

  4. utensil : fork

  5. smock : instrument

Solution:An ATOLL is a type of ISLAND: a small island formed by a coral reef. Similarly, a FORK is a type of UTENSIL. The answer is (D). The eye-catcher is (A) since an ARCHIPELAGO is a string of islands.
