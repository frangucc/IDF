Q.   If x + y = 4a/5, y + z = 7a/5 and z + x = 9a/5, then x + y + z =

  1. 7a/15

  2. a

  3. 2a

  4. 3a

  5. 4a

Solution:Writing the system of given equations vertically yields

x + y = 4a/5

y + z = 7a/5

z + x = 9a/5

Adding the three equations yields

(x + y) + (y + z) + (z + x) = 4a/5 + 7a/5 + 9a/5

2x + 2y + 2z = 20a/5 by adding like terms

2(x + y + z) = 4a

x + y + z = 2a by dividing both sides by 2
The answer is (C).
