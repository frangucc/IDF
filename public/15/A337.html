Q. Four letters a, b, c, and d represent one number each from one through four. No two letters represent the same number. It is known that c > a and a > d. If b = 2, then a =

  1. 1

  2. 2

  3. 3

  4. 4

  5. Not enough information to decide.

Solution: Combining the inequalities c > a and a > d gives c > a > d. Since b = 2, a, c, and d must represent the remaining numbers 1, 3, and 4 not necessarily in that order. In order to satisfy the condition c > a > d, c must be 4, a must be 3, and d must be 1. The answer is C.
