Q.   If w widgets cost d dollars, then at this rate how many dollars will 2000 widgets cost?

Solution:Most students struggle with this type of problem, and the GRE considers them to be difficult. However, if you can identify whether a problem is a direct proportion or an inverse proportion, then it is not so challenging. In this problem, as the number of widgets increases so does the absolute cost. This is a direct proportion, and therefore we set ratios equal:

 w 
 d 
 = 
 2000 
 x 

Cross multiplying yields

w × x = 2000 × d

Dividing by w yields

x = 
 2000d 
 w 

The answer is (C).
