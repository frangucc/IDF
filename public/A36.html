Q. For all a and b, define a # b to be −√(a + b)2

Column A
2 # 3

Column B
2 – 3

  1. Column A is larger

  2. Column B is larger

  3. Column A and Column B are equal

  4. Not enough information to decide.

Solution:2 # 3 = −√(2 + 3)2 = −√52 = −√25 = −5, and 2 – 3 = –1.
Hence, Column B is larger. The answer is (B).
