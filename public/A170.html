Q. Determine which of the two expressions below is larger, whether they are equal, or whether there is not enough information to decide.

x is both an integer and greater than 1

Column A
2x+2

Column B
3x

  1. Column A is larger

  2. Column B is larger

  3. Column A and Column B are equal

  4. Not enough information to decide.

Solution: If x = 2, we get

Column A

2x + 2 = 22 + 2 = 24 = 16


Column B

3x = 32 = 9

In this case, Column A is larger. If x = 3, we get

Column A

2x + 2 = 23 + 2 = 25 = 32


Column B

3x = 33 = 27

In this case, Column A is again larger. If x = 4, we get

Column A

2x + 2 = 24 + 2 = 26 = 64


Column B

3x = 34 = 81

Now, Column B is larger. This is a double case, and therefore the answer is (D).
