Q. What is the perimeter of Triangle ABC In the figure below?

  1. 5 + √2 + √29

  2. 5 + 2√2 + √29

  3. 5 + 4√2 + √29

  4. 3√2 + √34

  5. 4√2 + √34

Solution: Point A has coordinates (0, 2), point B has coordinates (2, 0), and point C has coordinates (5, 4). Using the distance formula to calculate the distances between points A and B, A and C, and B and C yields

AB = √(0 − 2)2 + (2 − 0)2 = √4 + 4 = √8 = 2√2

AC = √(0 − 5)2 + (2 − 4)2 = √25 + 4 = √29

BC = √(2 − 5)2 + (0 − 4)2 = √9 + 16 = 5

Adding these lengths gives the perimeter of Triangle ABC:

AB + AC + BC = 2√2 + √29 + 5
The answer is B.
