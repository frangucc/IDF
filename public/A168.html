Q. Determine which of the two expressions below is larger, whether they are equal, or whether there is not enough information to decide.

x is a nonnegative integer

Column A
(0.6)x

Column B
1/3

  1. Column A is larger

  2. Column B is larger

  3. Column A and Column B are equal

  4. Not enough information to decide.

Solution: Let’s substitute the numbers 0, 1, 2, 3 into the expression (0.6)x

If x = 0, then (0.6)x = (0.6)0 = 1. In this case, Column A is larger.

If x = 1, then (0.6)x = (0.6)1 = 0.6. In this case, Column A is still larger.

If x = 2, then (0.6)x = (0.6)2 = 0.36. In this case, Column A is once again larger.

If x = 3, then then (0.6)x = (0.6)3 = 0.216. In this case, Column B is now larger.

Hence, there is not enough information, and the answer is (D).
