Q.   If the value of x quarters is equal to the value of x + 32 nickels, x =

  1. 8

  2. 11

  3. 14

  4. 17

  5. 20

Solution:The value of the x quarters is 25x, and the value of the x + 32 nickels is 5(x + 32). Since these two quantities are equal, we get

25x = 5(x + 32)

25x = 5x + 160

20x = 160

x = 8
The answer is (A).
