Q. Choose the lettered pair that best expresses a relationship similar to that expressed in the original pair.

STATE : CONFEDERACY ::

  1. apple : tree

  2. return address : envelope

  3. binoculars : sight

  4. velocity : acceleration

  5. soldier : army

Solution:A CONFEDERACY is made up of STATES. Likewise, an ARMY is made up of SOLDIERS. The answer is (E).
