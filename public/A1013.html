Q. Choose the lettered pair that best expresses a relationship similar to that expressed in the original pair.

FOIBLE : BASE ::

  1. benefit : disadvantageous

  2. inhumanity : insouciant

  3. tatters : worn

  4. retention : vexatious

  5. abandon : belligerent

Solution:A FOIBLE is a minor character flaw. To be BASE is to have a severely deficient character. Similarly, TATTERED clothing is severely WORN. The answer is (C).
